# Almacén de repuestos - Laravel + Vue 3

Proyecto fullstack en desarrollo orientado a la gestión de repuestos industriales.
Desarrollado como parte de mi transición profesional hacia el desarrollo de software a entornos industriales y automatización.

---

## Objetivo del proyecto

Construir una aplicación web moderna para la gestión de repuestos que permita:

- Crear, editar y eliminar repuestos.
- Controlar stock actual y mínimo.
- Registrar retiradas de material.
- Preparar base para histórico de movimientos.
- Evolucionar hacia arquitectura API + SPA.

El proyecto está diseñado con mentalidad escalable y buenas prácticas de desarrollo.

---

## Stack Tecnoógico

### Backend

- Laravel (API REST).
- PHP 8+.
- MySQL.
- Validaciones con FormRequest.
- Arquitectura MVC.

### Frontend

- Vue 3.
- Vite.
- SPA integrada en Laravel.

### Dev Tools

- Git & GitHub.
- Control de versiones profesional.
